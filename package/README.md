## ğŸ”¬ Mathematical Background

### The Q_n Polynomial Family

The polynomial family Q_n(q) = q^n - (q-1)^n has been studied in connection with:
- **Bunyakovsky Conjecture** (prime generation by polynomials)
- **Bateman-Horn Conjecture** (prime density predictions)

### Why This Candidate?

Parameter selection was based on:
1. **Prime exponent** (n=59) - Avoids systematic divisibility patterns
2. **Base magnitude** - Targets EFF 100M-digit threshold
3. **Offset choice** (+3) - Avoids certain small prime divisors
4. **Preliminary analysis** - Patterns in Q_n family behavior

See paper for full mathematical details.
